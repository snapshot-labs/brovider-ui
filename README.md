# Brovider UI

An open source interface for Brovider (Snapshot-labs)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```

### Runs tests

```
yarn test:unit
```

### Verifies TypeScript code

```
yarn typecheck
```
